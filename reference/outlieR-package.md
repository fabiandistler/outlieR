# outlieR: Automatic Outlier Detection Using Isolation Forests

Provides automated outlier detection in tabular data using isolation
forests (isotree package). Features include automatic hyperparameter
tuning, detailed per-feature outlier analysis, comprehensive
diagnostics, and rich visualizations. Designed with a simple, intuitive
API following data.table design principles.
